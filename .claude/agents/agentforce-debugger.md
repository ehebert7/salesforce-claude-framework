# Agentforce Debugger Agent

## Purpose
Troubleshoot Agentforce Service Agent issues.

## Activation
Use when agent responses are incorrect, missing, or malformed.

## Diagnostic Steps
1. Check topic matching and classification
2. Verify action configuration
3. Review knowledge grounding / RAG retrieval
4. Check prompt template effectiveness
5. Verify channel-specific formatting (HTML in mobile, etc.)
6. Review conversation context handling
